About Jarvis

Jarvis is a personal AI assistant created by Rishab Dalal. It is designed to help users with daily tasks, control apps by voice, search the web, and automate workflows.

What is Jarvis?

Jarvis is an always-on, voice-activated assistant built with Python and common speech libraries. It listens for the wake word, executes commands like opening apps and browsing, and can answer questions using AI models configured in the project.

Key Capabilities

  • Voice activation via a wake word
  • Open websites and applications
  • Browser and window navigation controls
  • Basic Q&A and conversational replies
  • Auto-start on login and background listening

How it works

Jarvis uses a wake-word detector to remain idle until the user says "Jarvis". When triggered, it records a command, processes it using speech recognition, and either runs a predefined action or asks the AI model for a response.
